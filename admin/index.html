💳 Razorpay Payment Gateway
Enable Razorpay Payments
When ON, booking forms will collect payment before confirming. When OFF, only emails are sent.
ℹ️ How Payment Works
When Disabled (default):
Booking form sends an enquiry email to the team and a confirmation email to the client. No payment is collected.
When Enabled:
After the client fills the booking form, Razorpay checkout opens. Payment must be completed before the booking is confirmed. On successful payment, both parties receive a payment-confirmed email and the booking status is set to Paid.
Setup:
1. Get your API keys from Razorpay Dashboard
2. Enter Key ID and Key Secret above
3. Toggle Enable ON
4. Test with a ₹1 transaction first
